Exercice : Créer une page "Futur Développeur IA"

Objectif :
Créer une page personnelle avec :
- Une photo de profil
- Un titre et une description
- Un petit mot dans un encadré
- Trois icônes avec texte (À propos, Email, Téléphone)
- Style propre, moderne et responsive
- HTML et CSS séparés

Étapes de l'exercice :

1. Préparer les fichiers :
   - Crée index.html et style.css
   - Lier CSS au HTML : <link rel="stylesheet" href="style.css" />

2. HTML de base :
<!doctype html>
<html lang="fr">
<head>
  <meta charset="utf-8" />
  <meta name="viewport" content="width=device-width, initial-scale=1" />
  <title>Futur Développeur IA — Ton Nom</title>
  <link rel="stylesheet" href="style.css" />
</head>
<body>
</body>
</html>

3. Ajouter la carte principale :
<article class="card">
  <aside class="photo-area"> ... </aside>
  <section class="content"> ... </section>
</article>

4. Ajouter la photo :
<div class="photo-placeholder">
  <img src="URL_DE_TA_PHOTO" alt="Photo de profil" />
</div>
<div class="change-note">Emplacement photo — remplace l'image par la tienne</div>

5. Ajouter le contenu texte :
<h1>Futur développeur IA</h1>
<p class="lead">
  Je m'appelle <strong>Ton Nom</strong>, collégien. Mon objectif est de devenir développeur IA.
</p>
<div class="bio">
  <p><strong>Petit mot :</strong> J'apprends la programmation et l'IA.</p>
</div>

6. Ajouter les icônes :
<div class="icons">
  <div class="icon-card">

    </div>
  </div>

    </div>

    </div>
  </div>
</div>
